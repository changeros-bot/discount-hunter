## 一、最高結論

`/v17` 是使用者的根本主頁，不是實驗頁，不是 Market 91 新流程頁，也不是紙上交易頁。

`/v17` 必須維持「美股DCA 折價追蹤」原始大卡片版。

原版特徵：

- 大型 Hero 卡片：**美股DCA 折價追蹤**
- 右上角：`V17-M`
- 顯示：`BINANCE XSTOCKS`
- 有 `SYNC` 狀態與時間
- 有「今日決策」
- BTC 決策卡
- D1 / D2 / D3 / D4 / D5 層級規則
- 有「已完成」「略過本層」按鈕
- 有「鏈上持倉」
- 有「持倉區」
- 有「觀察區」
- 有 State Machine 檢查

任何 AI 或開發者在未取得使用者明確同意前，禁止重寫、簡化、替換、重構或混入其他系統。

2026-07-09 曾錯誤地將 Market 91 Action Gate / 紙上交易 / 簡化版 Box UI 併入 `/v17`，導致原本已花一個月建立的折價獵人主頁被替換成錯誤版。

錯誤包含：

1. 將 `/v17` 改成簡化版「折價獵人」頁。
2. 將 Paper Trading 放入 `/v17`。
3. 將 Market 91 Action Gate 語言放入根本主頁。
4. 把原本的大卡片版 UI 覆蓋掉。
5. 未先確認使用者是否允許改動根本主頁。

使用者明確指出：

> 這是我的根本，我的基礎，不要亂改。

此句視為本專案最高保護規則之一。

## 四、禁止事項

以下檔案不得在未取得使用者明確確認前改動：

- `pages/v17.js`
- `components/v17-dashboard-ui.js`
- `lib/v17-state-classifier.js`
- 任何會改變 `/v17` 視覺結構或資料流的 API

以下內容不得混入 `/v17`：

- Market 91 候選池 UI
- 紙上交易 UI
- Action Gate 實驗頁 UI
- Auto Whitelist UI
- Semi-auto Draft UI
- Permission Dry-run UI
- Evidence Pilot UI
- 新版簡化 dashboard UI

`/v17` 不應被改稱為：

- Market 91 Action Gate
- Paper Trading
- 17檔主名單頁
- 現金與預算頁
- 自動化門口

它只能是：

> 美股DCA 折價追蹤 / 折價獵人 V17 主頁
