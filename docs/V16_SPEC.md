# DCA 折價獵人 V16 規格

更新日期：2026-06-25

## 1. 專案定位

DCA 折價獵人不是自動交易系統。

定位是：

- 長期 DCA + 逢低加碼決策系統
- 手機優先
- 使用者打開 App 後，應在 30 秒內知道今天是否該買、買哪一檔、買哪一層、買多少

資料來源：

- Binance xStocks / tokenized stocks 價格資料
- 鏈上 Wallet 持倉
- Buy Ledger
- Telegram Bot

## 2. 買入類型

每檔標的同時存在兩種買入模式。

### 2.1 DCA 定期買入

Ledger 代碼：`N`

用途：

- 固定時間投入
- 與跌幅無關
- 不影響 D1-D4 判斷

### 2.2 折價買入

Ledger 代碼：

- `D1`
- `D2`
- `D3`
- `D4`

用途：

- 依 52 週高點或該商品可用最高點的跌幅觸發
- 分層買入
- 每一層獨立登帳

## 4. Today Decision 公式

今日決策只顯示：

```text
目前已觸發層級 - Ledger 已登帳層級 = 今日決策
```

例如 RKLBon：

- D1 已登帳
- D2 已登帳
- D3 已達但未登帳

今日決策只顯示：

```text
RKLBon D3
```

不再顯示 D1 / D2。

## 5. Ledger 定義

Ledger 是歷史紀錄與去重依據，不代表目前價格在哪一層。

Ledger 必須記錄：

- N
- D1
- D2
- D3
- D4

每筆至少包含：

- time
- amount
- price
- mode
- note
- leftBuyZone
- leftBuyZoneAt

## 6. 價格區間是動態的

價格所在區間由最新跌幅決定。

例如 RKLBon：

- D1 = -25%
- D2 = -40%
- D3 = -60%

若目前 -42%，畫面應顯示：

```text
D2 → D3
```

若反彈到 -30%，畫面應退回：

```text
D1 → D2
```

Ledger 仍保留 D1 / D2 歷史，但目前狀態要跟價格動態同步。

## 7. 進度條：水壺模型

進度條是動態水位。

- 跌越深，百分比越高
- 反彈時，百分比下降
- 回到上一區間，進度條也退回上一段

例如：

```text
D1=-25%
D2=-40%
目前=-37%
```

進度：

```text
(37 - 25) / (40 - 25) = 80%
```

100% 的意義：

- 到達下一層買點
- 該層進入今日決策
- Telegram 可發出新增層級提醒

## 8. 同層重新觸發規則

同一標的同一層預設只買一次。

要重新開放同層，必須同時滿足：

1. 價格先離開該買點區
2. 離開買點區後超過 24 小時
3. 再次跌回該買點區

24 小時應以 `leftBuyZoneAt` 計算，而不是買入時間。

## 9. Gap Down 跳空規則

若價格從非買點區直接跌穿多層，例如：

```text
0% → -43%
```

且該標的是：

```text
D1=-25%
D2=-40%
D3=-60%
```

則今日決策應同時列出：

```text
D1
D2
```

不得只列最深層。

## 10. Reconcile 補登規則

Wallet 可用來輔助補登 Ledger，但不能直接取代 Ledger。

補登時應以最新價格與目前已觸發層級為準，不追認過去錯過的價格。

補登 D1-D4 時，必須檢查：

```text
Wallet 總成本 - Ledger 已登帳成本 >= 該層建議金額
```

只有成本差額足夠，才補該層，避免只買 D1 卻誤補 D2。

## 11. Telegram 規則

Telegram 只通知新增層級。

規則：

- 同標的同層 12 小時最多提醒一次
- 已登帳後停止提醒
- 價格停留同層不重複提醒
- 只有新增層級、重新進入且符合 24 小時規則時才提醒

## 12. 首頁正確區塊

首頁應分為：

1. 今日決策：只顯示未登帳、需要操作的買點
2. 買點區標的：目前仍在 D1-D4 區間內的標的卡片
3. 觀察區：尚未進入 D1 的標的
4. Wallet 持倉
5. Ledger 檢查

不應再使用容易混淆的「買點警報」區塊。

## 13. 封版條件

V16 封版前必須通過：

- DCA/N 與 D1-D4 分帳
- Gap Down 多層列出
- 已登帳層級從今日決策消失
- 價格反彈時進度條動態退回
- 同層重新觸發符合離開區間 + 24 小時 + 重新進入
- Wallet 補登不誤補更深層
- Telegram 只通知新增未登帳層級
